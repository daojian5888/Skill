# 线索管理泳道图示例

这个示例展示 `swimlane-diagram` 在企业管理流程场景下的典型输出。

## 场景

围绕线索管理流程，按角色拆分泳道，展示从线索获取、线索分配、销售跟进、客户拜访，到线索盘点与培育淘汰的关键路径和分支。

## 示例图

![线索管理泳道图](./lead-management-swimlane.svg)

## 适合观察的点

- 纵向泳道标题栏是否清晰
- 主流程是否稳定向下
- 判断节点与动作节点是否区分明确
- 横向交接是否克制
- 数据池图形是否规范

## 推荐提示词

- `用 $swimlane-diagram 把线索管理流程做成标准 SVG 泳道图`
- `用 $swimlane-diagram 按企业管理模板生成线索跟进流程图`
- `用 $swimlane-diagram 参考正式业务流程图风格输出成品图`

## 说明

这个案例用于展示成品图效果。后续如果继续扩展仓库，建议为合同管理、采购管理、交付管理、售后管理分别补一组示例，这样更利于传播和涨星。
