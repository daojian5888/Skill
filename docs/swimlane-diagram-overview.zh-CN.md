# swimlane-diagram 技能简介

`swimlane-diagram` 是一个面向企业管理与业务流程场景的标准泳道图技能。

它的目标不是输出“像图的代码”，而是直接输出正式可展示的流程图成品。默认交付物是 `SVG`，只有在用户明确要求时，才会降级输出 `PlantUML` 或 `Mermaid`。

## 适用场景

- 企业管理流程
- SOP / 操作规范
- 审批流
- 合同流
- 采购流
- 交付流
- 客户流程
- 跨部门协同流程

## 优势

- 默认输出 `SVG` 成品图，适合正式文档
- 风格更接近 draw.io / 业务流程图，而不是技术草图
- 约束更严格，强调节点对齐、文字居中、箭头贴边和少折角连线
- 更适合持续沉淀成统一风格的流程图体系
