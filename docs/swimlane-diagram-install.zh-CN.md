# swimlane-diagram 安装与使用

## 安装路径

这是一个技能仓库。安装时请指向技能子目录，而不是仓库根目录：

- `skills/swimlane-diagram`

仓库地址：

- `https://github.com/daojian5888/Skill`

## 使用示例

- `用 $swimlane-diagram 把这个流程做成标准 SVG 泳道图`
- `用 $swimlane-diagram 按企业管理模板生成成品图`
- `用 $swimlane-diagram 输出 PlantUML 版本`
- `用 $swimlane-diagram 输出 Mermaid 版本`

## 建议

- 如果目标是正式展示、制度文档或培训材料，优先使用默认 `SVG`
- 如果目标是后续人工维护源码，再明确要求 `PlantUML` 或 `Mermaid`
