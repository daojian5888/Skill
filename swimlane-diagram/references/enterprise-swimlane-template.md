# Enterprise Swimlane Template

Use this template when the user wants a management-process swimlane similar to a draw.io business chart.

## Default Use Case

Best for:

- enterprise management systems
- approval processes
- lead / customer / contract workflows
- delivery and service workflows
- HR / admin / finance internal processes

## Required Visual Standard

- vertical lane columns
- fixed header band
- one dominant top-to-bottom reading path
- equal-width action nodes within the same diagram
- decision diamonds centered on the lane axis
- branch labels offset from shapes and connectors
- connectors end at the edge of shapes
- no line should run through text

## Node Family Standard

### Start / End

- terminator or pill shape
- width consistent within the pair

### Action

- rounded rectangle
- same width for all main action nodes
- same height for all main action nodes

### Decision

- diamond
- centered text
- short question only

### Data / Pool / Store

- standard database cylinder only
- single fill color
- top ellipse visible
- side walls vertical
- bottom represented as front-half curve only

Do not fake the data-store shape with unrelated geometry.

## Connector Standard

- main path: vertical inside a lane
- handoff: horizontal between lanes
- branch path: horizontal then vertical
- keep bends minimal
- do not create decorative detours

## Final QA Checklist

Before considering the diagram complete, verify:

- labels centered
- comparable nodes aligned
- branch labels readable
- no connector crossing text
- data-store symbol not inverted
- no mixed fill tones inside one symbol unless explicitly intended
- overall visual style consistent
