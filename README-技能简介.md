# swimlane-diagram 技能简介

`swimlane-diagram` 是一个面向企业管理、SOP、审批流、交付流和跨部门协同场景的标准泳道图技能。

它的核心特点不是“能生成流程图”，而是：

- 默认输出正式 `SVG` 成品图
- 优先适配企业管理和业务流程场景
- 风格更接近 draw.io / 正式业务流程图
- 内置版式和质量约束，减少不同线程下的输出漂移
- 同时保留 PlantUML / Mermaid 作为兼容型源码输出

## 适合场景

- 招聘、人事、资产、档案等行政管理流程
- 合同、采购、仓储、回款等商务流程
- 售前、交付、验收、售后等服务流程
- 线索、商机、客户、报价等营销流程
- 各类审批流、协同流、制度流程

## 核心优势

1. 默认交付成品图，而不是伪泳道源码
2. 更适合管理文档、制度手册、培训材料和知识库
3. 强调纵向泳道、标题栏、节点对齐、箭头贴边、分支简洁
4. 支持长期沉淀统一风格的企业流程图体系

## 使用示例

- `用 $swimlane-diagram 把这个流程做成标准 SVG 泳道图`
- `用 $swimlane-diagram 按企业管理模板生成成品图`
- `用 $swimlane-diagram 输出 PlantUML 版本`
