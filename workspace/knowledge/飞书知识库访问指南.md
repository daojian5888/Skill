# 飞书知识库访问总结

## 工具组合

1. **feishu_wiki_space** (list) → 获取所有知识空间
2. **feishu_wiki_space_node** (list, space_id=xxx) → 列出空间内所有节点
3. **feishu_wiki_space_node** (get, token=xxx) → 获取节点详情（含 obj_token）
4. **feishu_fetch_doc** (doc_id=obj_token) → 读取实际文档内容
