# 道简网络三大系统深度理解

## 一、DJOS vs DJMS vs DJAS 本质区别

| 系统 | 全称 | 本质 | 类比 |
|------|------|------|------|
| **DJOS** | 道简网络操作系统 | 流程标准 - 告诉员工"怎么干" | "**道**" |
| **DJMS** | 道简网络管控系统 | 管理制度 - 告诉管理者"怎么管" | "**法**" |
| **DJAS** | 道简网络应用系统 | 软件系统 - 给员工"用什么干" | "**术**" |

---

## 二、核心理解：流程 = 经验沉淀和复制

### 核心理念
- **"人可以走，经验留下"** - 把优秀员工的大脑变成组织资产
- 流程让平凡的人也能做出不平凡的事
- 实现企业的"自动复制" - 像麦当劳一样快速拓展

### 流程的目的
```
坏的流程 → 让好人做不了好事
好的流程 → 让坏人做不了坏事
```

### 模式化 = 标准化 + 模块化 + 流程化
- 标准化：统一标准
- 模块化：拆分独立模块
- 流程化：串联成线

## 三、DJOS 营销业务管理制度（2025Q3）详解

### 1. 日常管理
| 项目 | 要求 |
|------|------|
| 日常签到 | 外勤签到，按员工管理制度 |
| 日报制度 | 当天24:00前提交（今日工作、明日计划、其他事项） |
| 夕会 | 每天17:00，复盘当天工作 |
| 周例会 | 每周一9:30，日常宣贯+培训+复盘 |

### 2. 线索管理
- **线索来源**：公司资源、协会信息、线上数据、客户转介绍、厂商信息、市场活动、个人挖掘
- **分配**：营销部经理每周分配+复盘
- **跟进时效**：72小时内必须联系，3次无法联系或2周无法拜访需退回
- **转化时限**：15天未转化返回线索池，30天内不得再次领取
- **上限**：每人最多50条线索

### 3. 客户管理
**客户分级**：
| 级别 | 上限 | 拜访频率 |
|------|------|----------|
| 战略客户 | 10家 | 每两周1次 |
| 重要客户 | 20家 | 每月1次 |
| 普通客户 | 50家 | 每季度1次 |
| 低价值客户 | 20家 | 谨慎投入 |

- **公海机制**：180天未成交移至公海
- **服务群**：战略/重要客户必须建服务群

### 4. 外勤管理
- **有效外勤条件**：
  1. 精确地理位置+照片
  2. 会见客户联系人
  3. 详细沟通记录
- **考核**：每月50次满分，1000元奖金

### 5. 项目管理
- **信息录入**：当日内录入系统，变更需当日更新
- **毛利率要求**：
  - 纯货物：≥5% 才提成
  - 含施工/客户费用：≥10% 才提成

### 6. 报价管理
- **时限**：1小时内完成（询价2小时内）
- **选型限制**：不超过3种
- **成本**：实际成本为准，库存产品每月1%折算

### 7. 报备管理
- 所有项目型分销产品需报备
- 2小时内发起，24小时内更新结果

### 8. 合同管理
- 首次交易不得提供账期
- 账期最长5个月
- 需总经理审批的情况：首次交易账期、20万以上合同

### 9. 订单管理
- 销售订单需商务经理+产品经理审批
- 编辑权限仅商务经理

### 10. 发货管理
- 商务经理当日完成发货
- 紧急要货需群内报备审批

### 11. 回款管理
| 逾期天数 | 处罚 |
|----------|------|
| 30天 | 提成减半 |
| 60天 | 取消提成，工资抵扣 |
| 累计3次逾期 | 以后款到发货 |

- **账期成本**：每30天1%

### 12. 激励管理
| 岗位 | 提成比例 | 门槛 |
|------|----------|------|
| 销售顾问 | 实际毛利20% | 毛利率≥5% |
| 电商销售 | 实际毛利10% | 毛利率≥5% |
| 商务经理 | 实际毛利1% | 毛利率≥5% |
| 产品经理 | 实际毛利1% | 毛利率≥5% |

## 四、DJMS（管控系统）核心红线

1. **采购权责**：只能商务经理采购，严禁私自联络供应商（红线，违者开除）
2. **供应商**：核心供应商优先，三家比价
3. **品牌**：二线及以上正规品牌，严禁白牌/三无
4. **发票**：明细必须与订单完全一致
5. **账期**：最长3个月，需总经理同意
6. **回款逾期**：严重者取消提成
