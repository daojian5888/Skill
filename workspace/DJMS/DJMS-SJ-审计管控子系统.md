## 1. 审计管控子系统概述

### 1.1 定位与职责

审计管控子系统是道简网络运行系统（DJRS）管控层的**独立监督保障**，承担着独立评价、监督、鉴证和改进的功能，是 DJMS 五大子系统中唯一具有独立性的管控模块，以"第三道防线"的视角为管理层提供客观公正的风险与控制信息。

**核心职责**：
- 建立并维护道简网络内部审计体系
- 开展财务审计（财务报告/资金/资产/负债）
- 开展运营审计（业务流程/效率/合规）
- 开展专项审计（项目审计/离任审计/合规审计）
- 管理审计整改跟踪，推动问题闭环
- 建立反舞弊机制，管理举报投诉
- 对接 DJMS-FX 风险管控子系统，为风险管控提供独立评估
- 对接外部审计（年度审计/专项审计）

**管控边界**：
- 内部审计的独立评价与监督
- 审计发现与整改跟踪
- 反舞弊管理与举报处理
- 外部审计协调与管理

### 1.2 模块架构

审计管控子系统包含以下九个核心模块：

| 序号 | 模块名称 | 编码 | 核心价值 |
|------|----------|------|----------|
| 1 | 内部审计管理 | 5.1.1 | 独立监督保障 |
| 2 | 财务审计 | 5.1.2 | 财务合规鉴证 |
| 3 | 运营审计 | 5.1.3 | 运营效率提升 |
| 4 | 项目审计 | 5.1.4 | 项目管控保障 |
| 5 | 离任审计 | 5.1.5 | 责任明确追溯 |
| 6 | 合规审计 | 5.1.6 | 合规风险揭示 |
| 7 | 审计整改跟踪 | 5.1.7 | 问题闭环落实 |
| 8 | 反舞弊管理 | 5.1.8 | 廉洁文化守护 |
| 9 | 外部审计协调 | 5.1.9 | 外部审计对接 |

## 3. 各模块详细流程

### 3.1 内部审计管理（5.1.1）

#### 3.1.1 模块职责

- 建立并维护道简网络内部审计章程与制度体系
- 制定年度审计计划并推动执行
- 管理内部审计项目（立项/执行/报告/归档）
- 协调外部审计资源（会计师事务所/律师事务所）
- 管理审计档案与知识沉淀
- 提升内部审计能力与专业水平
- 对审计委员会/管理层负责，确保审计独立性

#### 3.1.2 一级流程：内部审计管理

```plaintext
【内部审计管理一级流程】

风险评估与审计计划制定
    │
    ├─ 收集风险信息（DJMS-FX/业务部门/历史审计）
    ├─ 风险排序与审计重点确定
    ├─ 制定年度审计计划
    └─ 审计计划审批（审计委员会/管理层）
    │
    ↓
审计资源配置
    │
    ├─ 审计人员安排
    ├─ 审计时间计划
    └─ 审计工具与方法
    │
    ↓
审计项目执行
    │
    ├─ 项目立项（确定审计范围/目标/团队）
    ├─ 审计通知（提前3-5天通知被审计单位）
    ├─ 现场审计
    ├─ 审计发现沟通
    ├─ 审计报告编制
    └─ 审计报告审批与发布
    │
    ↓
审计整改跟踪
    ↓
审计归档与知识沉淀
```

#### 3.1.3 年度审计计划制定流程

```plaintext
每年11月启动下年度审计计划制定
    ↓
收集审计需求
    │
    ├─ 管理层审计需求
    ├─ 审计委员会指示
    ├─ DJMS-FX风险管控子系统风险清单
    ├─ 监管合规要求（年度审计/行业要求）
    └─ 历史审计发现问题
    ↓
风险评估与优先级排序
    │
    ├─ 高风险领域（舞弊风险/合规风险/重大风险）
    ├─ 中风险领域（运营效率/财务管理）
    └─ 低风险领域（一般管理领域）
    ↓
确定审计项目清单
    │
    ├─ 必做项目（年度例行审计）
    └─ 选做项目（根据风险评估排序）
    ↓
资源配置与时间安排
    ↓
审计计划审批
    ↓
计划发布与执行
```

| 审计类型 | 频次 | 覆盖范围 | 审计周期 |
|----------|------|----------|----------|
| 财务审计 | 年度 | 全公司 | 每年1-3月 |
| 运营审计 | 半年 | 各部门 | 上下半年各1次 |

| 专项审计 | 按需 | 举报/特定事项 | 按需启动 |

### 3.2 财务审计（5.1.2）

#### 3.2.1 模块职责

- 审计道简网络财务报告的真实性、完整性和准确性
- 审计资金管理（现金/银行存款/票据）
- 审计资产管理（固定资产/无形资产/存货）
- 审计负债管理（应付账款/其他应付款）
- 审计成本费用管理（成本核算/费用报销）
- 审计税务合规（税务申报/发票管理）
- 对接 DJMS-JS 计划预算子系统的财务数据

#### 3.2.2 一级流程：财务审计

```plaintext
【财务审计一级流程】

审计计划与准备
    │
    ├─ 确定审计范围（全面/专项）
    ├─ 组建审计团队
    ├─ 收集财务资料
    └─ 制定审计程序
    │
    ↓
现场审计
    │
    ├─ 内部控制测试
    ├─ 实质性测试（账务/凭证/报表）
    ├─ 资产盘点（现金/固定资产）
    └─ 函证（银行/往来）
    │
    ↓
审计发现
    │
    ├─ 识别审计问题
    ├─ 评估问题严重程度
    └─ 与被审计单位沟通
    │
    ↓
审计报告
    │
    ├─ 编制审计报告
    ├─ 审计发现分级（重大/重要/一般）
    └─ 管理建议
    │
    ↓
报告审批与发布
    ↓
整改跟踪
```

#### 3.2.3 财务审计关键审计领域与程序

| 审计领域 | 审计重点 | 审计程序 | 关键风险 |
|----------|----------|----------|----------|
| 现金管理 | 账实相符/使用合规 | 现金盘点/银行对账 | 坐支现金/小金库 |
| 银行存款 | 账户完整/余额真实 | 银行函证/对账单核对 | 账外账户/资金挪用 |
| 应收账款 | 账龄准确/可收回性 | 账龄分析/函证/催收检查 | 虚增收入/坏账不实 |
| 固定资产 | 账实相符/折旧准确 | 资产盘点/折旧测试 | 资产流失/折旧错误 |
| 应付账款 | 完整性/准确性 | 供应商对账/发票核对 | 漏记负债/虚假付款 |
| 成本核算 | 成本归集合理/准确 | 成本计算复核/完工对比 | 成本虚增/利润操纵 |
| 费用报销 | 真实性/合规性 | 费用抽查/审批复核 | 虚假报销/超标准报销 |
| 税务合规 | 申报准确/缴纳及时 | 税表核对/申报记录 | 偷逃税款/税务处罚 |

    ↓

    ↓

#### 3.3.3 运营审计关键领域

| 运营领域 | 审计重点 | 评估标准 | 常见问题 |
|----------|----------|----------|----------|
| 销售运营 | 销售流程/合同管理/客户管理 | 流程合规性/效率/效果 | 合同审批不规范/客户档案不完整 |
| 采购运营 | 供应商管理/采购流程/价格管理 | 采购合规性/成本控制/质量保障 | 供应商选择不透明/采购价格偏高 |

| 行政运营 | 资产管理/费用管理/后勤保障 | 资产安全/费用合规/服务满意度 | 资产闲置/费用超支 |
| IT运营 | 系统安全/数据管理/变更管理 | 安全性/可用性/合规性 | 系统漏洞/数据泄露/变更失控 |

### 3.5 离任审计（5.1.5）

- 评估离任干部任职期间重大决策的合规性与效果
- 识别离任交接中的资产/财务问题
- 为干部考核、任免提供审计依据

- 对接 DJMS-FX 风险管控子系统的合规风险清单
- 管理合规审计发现，推动整改

### 3.7 审计整改跟踪（5.1.7）

#### 3.7.1 模块职责

- 验证审计整改的实际效果
- 推动整改闭环（问题彻底解决）
- 对整改不力的情况进行升级处理

#### 3.7.2 一级流程：审计整改跟踪

```plaintext
【审计整改跟踪一级流程】

审计发现确认
    │
    ├─ 审计报告发布
    ├─ 审计发现分级（重大/重要/一般）
    └─ 整改要求明确
    │
    ↓
整改任务下发
    │
    ├─ 整改责任单位确认
    ├─ 整改责任人指定
    ├─ 整改期限确定
    │   ├─ 重大问题：30天内
    │   ├─ 重要问题：60天内
    │   └─ 一般问题：90天内
    └─ 整改验收标准明确
    │
    ↓
整改执行跟踪
    │
    ├─ 建立整改台账
    ├─ 定期跟踪（周/双周/月）
    ├─ 整改进度提醒
    └─ 问题协调（如需）
    │
    ↓
整改验收
    │
    ├─ 被审计单位提交整改报告
    ├─ 审计组验证整改效果
    └─ 验收通过/不通过处理
    │
    ↓
整改闭环确认
    ↓
归档与持续跟踪（复发性问题）
```

#### 3.7.3 审计发现问题分级与处理

## 4. 执行保障

### 4.1 监控指标表

| 监控指标 | 计算方式 | 频次 | 责任人 | 达标标准 |
|----------|----------|------|--------|----------|
| 审计计划完成率 | 已完成审计项目/计划项目×100% | 年度 | 审计负责人 | ≥90% |
| 审计报告及时率 | 按时出具报告数/应出具报告数×100% | 季度 | 审计负责人 | ≥95% |
| 审计整改完成率 | 已整改问题数/审计发现问题总数×100% | 半年度 | 审计负责人 | ≥85% |
| 重大问题整改及时率 | 重大问题及时整改数/重大问题总数×100% | 季度 | 审计负责人 | 100% |
| 举报处理及时率 | 按时处理举报数/举报总数×100% | 季度 | 审计负责人 | ≥90% |
| 反舞弊培训覆盖率 | 已培训人数/应培训人数×100% | 年度 | 审计负责人 | ≥95% |
| 外部审计配合满意度 | 外部审计师评分 | 年度 | 财务部 | ≥4分（5分制） |
| 审计建议采纳率 | 被采纳建议数/提出建议总数×100% | 年度 | 审计负责人 | ≥70% |

### 4.2 持续优化机制

1. **年度审计工作总结**：每年初对上年度审计工作进行总结，评估审计效果，优化审计方法
2. **季度审计例会**：每季度召开审计例会，跟踪审计整改进展，评估风险状况
3. **审计方法论更新**：定期引入先进审计方法论（风险导向审计/数据审计），提升审计效率
4. **审计能力建设**：组织审计人员参加专业培训（CIA/CPA），提升专业胜任能力
5. **最佳审计实践沉淀**：将优秀审计案例沉淀为审计手册，持续优化审计管控体系
